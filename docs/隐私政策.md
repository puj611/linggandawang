## 一、引言

欢迎使用"灵感大王"（以下简称"本软件"）。本软件是一款桌面悬浮窗工具，通过反向提问帮助用户将模糊灵感转化为精准的 AI 提示词。

我们深知个人信息对您的重要性，本隐私政策旨在向您说明：
- 我们收集哪些信息
- 我们如何使用这些信息
- 我们如何保护您的信息
- 您对信息享有的权利

**请您在使用本软件前仔细阅读本隐私政策。**

---

## 二、我们收集的信息

### 2.1 我们**不主动收集**的信息

| 信息类型 | 说明 |
|----------|------|
| 个人身份信息 | 姓名、邮箱、电话、地址等 |
| 设备信息 | 设备型号、序列号、IMEI 等 |
| 位置信息 | GPS、IP 地理位置等 |
| 使用行为 | 操作记录、点击行为等 |

### 2.2 本地存储的信息

以下信息**仅存储在您的设备本地**，不会上传到任何服务器：

| 信息类型 | 存储位置 | 用途 |
|----------|----------|------|
| 用户偏好设置 | 本地数据库（SQLite） | 记住您的配置（如快捷键、窗口位置） |
| 提示词历史 | 本地数据库（SQLite） | 保存您生成的提示词记录 |
| 题库缓存 | 本地数据库（SQLite） | 缓存题目数据，提升加载速度 |
| API Key | 本地数据库（SQLite） | 存储您配置的 AI 服务密钥 |

### 2.3 第三方服务

当您使用本软件的 AI 提示词生成功能时，您的输入内容会发送到您配置的 **第三方 AI 服务**（如 OpenAI、豆包等）：

| 数据 | 发送对象 | 说明 |
|------|----------|------|
| 您的输入文本 | 您配置的 AI 服务 | 用于生成提示词 |
| API Key | 您配置的 AI 服务 | 用于身份验证 |

**请注意**：我们不控制第三方 AI 服务的数据处理方式，请您查阅相应服务的隐私政策。

## 四、信息存储与安全

### 4.1 存储位置

- 所有数据存储在**您的本地设备**上
- 数据库文件位于应用数据目录中
- 不使用云服务器存储

### 4.2 安全措施

| 措施 | 说明 |
|------|------|
| 本地存储 | 数据不离开您的设备 |
| 最小权限 | 应用仅请求必要的系统权限 |
| 开源透明 | 核心代码可供审查 |

### 4.3 数据保留

- 您可以随时通过应用内设置删除本地数据
- 卸载应用时，建议手动删除残留数据
