## 一、打包前必做检查

### 1.1 环境检查

```powershell
# 确认以下工具已安装且版本正确
node --version      # 需要 v18+
npm --version       # 需要 v8+
rustc --version     # 需要 1.70+
cargo --version     # 随 rustc 安装
```

### 1.2 代码检查

| 检查项 | 操作 | 说明 |
|--------|------|------|
| TypeScript 编译 | `npm run build` | 确保没有编译错误 |
| 单元测试 | `npm run test` | 28个测试全部通过 |
| 控制台无报错 | `npm run dev` 后查看 | 无红色错误信息 |
| 删除调试代码 | 全局搜索 `console.log` | 删除或注释掉调试日志 |
| 删除测试数据 | 检查硬编码数据 | 确保没有测试用的假数据 |

### 1.3 版本号检查

```
版本号位置（需要统一修改）：
├── package.json          → "version": "0.1.0"
├── src-tauri/tauri.conf.json → "version": "0.1.0"
└── src-tauri/Cargo.toml  → version = "0.1.0"

三个文件的版本号必须一致！
```

## 二、打包流程

### 2.1 使用打包脚本（推荐）

```powershell
# 方式1：使用 release 脚本（在普通 PowerShell 中运行）
cd "d:\AI工作平台\灵感大王"
.\build-release.ps1

# 方式2：使用简单脚本
.\build-simple.ps1
```

> **注意**：不要在 Trae 的终端中运行，需要在普通 PowerShell 窗口中执行！

### 2.2 手动打包

```powershell
# 1. 安装依赖
npm install

# 2. 构建前端
npm run build

# 3. 打包 Tauri 应用
npx tauri build
```

### 2.3 打包产物位置

```
打包完成后，安装包在以下目录：

src-tauri\target\release\bundle\
├── nsis\                     # NSIS 安装包（推荐）
│   └── linggandawang_0.1.0_x64-setup.exe
└── msi\                      # MSI 安装包（备选）
    └── linggandawang_0.1.0_x64_en-US.msi
```

## 三、打包常见问题

### 问题1：打包超时或卡住

```
原因：Rust 编译时间较长（首次约5-10分钟）
解决：
1. 耐心等待，不要中途取消
2. 如果卡死，关闭终端重新运行
3. 清理缓存：删除 src-tauri\target 目录后重试
```

### 问题2：提示"操作成功完成"但没有安装包

```
原因：在 Trae 终端中运行，沙箱环境限制
解决：在普通 PowerShell 窗口中运行打包脚本
```

### 问题3：找不到 rustc 或 cargo

```
原因：Rust 未安装或环境变量未配置
解决：
1. 安装 Rust：https://rustup.rs/
2. 重启终端
3. 验证：rustc --version
```

### 问题4：前端构建失败

```
原因：TypeScript 编译错误
解决：
1. 运行 npm run build 查看具体错误
2. 修复错误后重新打包
```

### 问题5：图标不生效

```
原因：图标格式或路径问题
解决：
1. 确保 icon.ico 是 256x256 像素
2. 确保路径在 tauri.conf.json 中正确配置
3. 清理缓存后重新打包
```

## 六、代码签名（可选但建议）

### 6.1 为什么需要签名？

| 未签名 | 已签名 |
|--------|--------|
| Windows 弹出"未知发布者"警告 | 显示公司/开发者名称 |
| 用户可能不敢安装 | 用户信任度更高 |
| 部分杀毒软件可能误报 | 减少误报 |

### 6.2 如何获取签名证书

```
国内证书提供商：
1. 沃通（wosign.com）- 约 300-500 元/年
2. 中国金融认证中心（CFCA）
3. JoySSL

国际证书提供商：
1. DigiCert - 约 $200/年
2. Sectigo - 约 $70/年
3. SSL.com - 约 $50/年
```

### 6.3 Tauri 应用签名方法

```powershell
# 签名命令（需要先获取证书）
npx tauri build -- --sign "证书名称"
```
