# 灵感大王 - 演示视频脚本

> **总时长**：4 分钟  
> **录制工具**：Win + G（Xbox Game Bar）或 OBS Studio  
> **分辨率**：1920x1080  
> **录制区域**：灵感大王悬浮窗 + 周围桌面（展示悬浮窗特性）

---

## 开场（0:00 - 0:25）

### 画面
- 灵感大王悬浮窗停留在桌面右上角，半透明状态
- 背景是一个代码编辑器（VS Code / Trae），打开一个有明显 UI 问题的页面

### 旁白
> "你有没有遇到过这种情况——打开 AI 编程工具，想改个页面，但脑子里只有'丑'这一个字？
> 
> AI 问你具体要改什么，你说不上来；勉强说了几句，AI 出来的东西完全不是你想要的。
> 
> 问题不在 AI，而在你给它的提示词太模糊了。
> 
> 这就是灵感大王要解决的事。"

---

## 功能一：三档模式（0:25 - 1:15）

### 画面 1（0:25 - 0:40）：展示模式选择器
- 点击灵感大王悬浮窗，展开主面板
- 鼠标依次悬停三个模式 Tab：「直接生成」「快速提问」「详细诊断」
- 每个 Tab 悬停时高亮

### 旁白
> "灵感大王提供了三档模式。
> 赶时间？选'直接生成'，一句话出提示词。
> 有方向？选'快速提问'，只问 7 个关键问题。
> 没想好？选'详细诊断'，31 道题帮你理清所有细节。"

### 画面 2（0:40 - 1:15）：演示直接生成
- 输入框输入：`给列表页加分页功能`
- 点击「直接生成」Tab
- 点击「直接生成 ->」按钮
- 等待生成，展示四段式提示词结果
- 鼠标滚动展示四段内容

### 旁白
> "比如我现在只想要一个分页功能，不需要那么多提问。
> 输入需求，选'直接生成'，点一下——
> 三秒钟，一份结构化的提示词就出来了。
> 它自动分成了'要做什么、怎么做、不能怎么、怎么算做好了'四段，
> 直接复制粘贴到 AI 工具里就能用。"

---

## 功能二：快速提问（1:15 - 2:15）

### 画面 1（1:15 - 1:30）：切换到快速提问
- 点击「重新提问」
- 输入：`页面太丑了，想改好看`
- 确认「快速提问」模式已选中
- 点击「开始提问 ->」

### 旁白
> "如果需求稍微复杂一点，可以用'快速提问'模式。
> 我输入'页面太丑了，想改好看'——"

### 画面 2（1:30 - 1:50）：快速回答 3-4 道题
- 第 1 题：点击「修一个看不顺眼的地方」
- 第 2 题：点击「颜色太多太花哨」
- 第 3 题：点击「极简克制（像 Apple/Linear）」
- 每次回答后底部标签增加，进度条推进

### 旁白
> "它只问最关键的问题。
> 第一题：你想改什么？
> 第二题：具体是哪方面的问题？
> 第三题：想要什么风格？
> 每个问题都有现成的选项，点一下就行，不需要打字。"

### 画面 3（1:50 - 2:15）：展示结果
- 进入结果态，展示四段式提示词
- 滚动查看内容
- 点击「复制」按钮（变绿）
- 点击「导出 .md」

### 旁白
> "7 个问题回答完，提示词就生成了。
> 比直接生成的版本更精准，因为它知道了你具体的痛点和风格偏好。
> 一键复制，或者导出成 Markdown 文件，随时可以用。"

## 功能三：截图诊断（2:15 - 3:00）

### 画面 1（2:15 - 2:30）：粘贴截图
- 回到展开态
- 在截图诊断区域 Ctrl+V 粘贴一张 UI 截图
- 显示"正在分析截图..."spinner

### 旁白
> "灵感大王还有一个杀手级功能——截图诊断。
> 把你的页面截图粘贴进来，它会自动分析。"

### 画面 2（2:30 - 2:50）：展示诊断结果
- 分析完成，显示诊断报告
- 鼠标指向各个问题项（对比度不足、间距不统一等）

### 旁白
> "它会告诉你：对比度不够、间距不统一、字号层级缺失……
> 这些都是肉眼不容易量化的视觉问题。
> 一键插入到规格段，自动进入提问流程，
> 生成的提示词会包含这些具体的修复建议。"

### 画面 3（2:50 - 3:00）：标签可视化
- 展示底部意图标签区
- 鼠标悬停各个标签

### 旁白
> "所有你回答的信息都会变成意图标签，
> 你可以随时查看、删除不需要的标签，
> 删除后提示词会自动重新生成。"

## 录制注意事项

### 环境准备
1. 确保灵感大王开发服务器已启动（http://localhost:5173/）
2. 关闭桌面无关通知（避免弹窗干扰）
3. 准备好一张 UI 截图（用于截图诊断演示）
4. 准备好一张参照图（用于情绪板演示）
5. 浏览器全屏或接近全屏，确保灵感大王悬浮窗清晰可见

### 操作节奏
- 每次点击后**停顿 1 秒**，等待动画完成再继续
- 输入文字时**稍微放慢**，让观众看清内容
- 展示结果时**滚动慢一点**，给观众阅读时间

### 常见问题预处理
| 问题 | 解决方案 |
|------|----------|
| LLM 未配置 | 提前配置好 DeepSeek API Key |
| 生成超时 | 准备好网络环境，确保 API 可用 |
| 截图诊断无结果 | 准备一张对比度低的截图，确保能检出问题 |
| 动画卡顿 | 关闭其他占用资源的应用 |

### 推荐截图素材
- 对比度低的页面截图（深色背景 + 深色文字）
- 间距不统一的列表页
- 字号层级不明显的仪表盘
